# EZDML（生成Erupt代码）

EZDML是一款国产免费的轻量级数据建模工具，具有轻巧、快速、简便而功能强大的特点，可轻松地进行数据库表结构设计，建立界面数据模型，支持通过自定义脚本模板生成测试数据、界面原型和前后端代码文件。
官网地址：[http://www.ezdml.com/](http://www.ezdml.com/)
