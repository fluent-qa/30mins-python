# 代码生成器 erupt-generator


## Generator
在导入erupt的前提下，pom.xml 加入如下依赖（1.6.1版本开始支持）
```xml
<!-- 代码生成器模块 -->
<dependency>
  <groupId>xyz.erupt</groupId>
  <artifactId>erupt-generator</artifactId>
  <version>${erupt.version}</version>
</dependency>
```
导入成功后重启即可看到代码生成相关菜单
![image.png](./img/Ufod_x-qGtb1mJw9/1609140227715-e00150bf-1797-49b7-aaa7-e88c6fb81ce5-384868.png)
![image.png](./img/Ufod_x-qGtb1mJw9/1609140258559-8e1515b8-16aa-4112-8bc6-e9937f3bad93-471667.png)

将生成的代码，复制或下载添加到项目中，重启项目，将类添加到菜单即可！
