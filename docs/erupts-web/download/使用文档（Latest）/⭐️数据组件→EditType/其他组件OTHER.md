# 其他组件 OTHER


## HIDDEN
**隐藏**
```java
@EruptField(
    edit = @Edit(
        title = "隐藏",
        type = EditType.HIDDEN
    )
)
private String input;
```


## EMPTY
**空（仍占据组件位置）**
```java
@EruptField(
    edit = @Edit(
        title = "空",
        type = EditType.EMPTY
    )
)
private String input;
```
